Doing Stuff with Web Things:

When you want to get the user's input or text area tag, you use the special word "value". When you want to show something to a user, you use section and the special word innerHTML.

  • Input or textarea => value
  • Section => innerHTML
  • key - value pairs => description - stuffpairs


Master Mobile UX:
  • Wherever you are doing - doing mobile first - address constraints of mobile size.
  • Desktop computers sales have been declining.
  • You don't see people with their heads in laptops.
  • Start small and scale up.
  • Mobile is not a fad.
  • Be there (to help someone along their decision)
  • Be useful
  • Be quick (make your interactions smooth and seamless, so the user doesn't even notice)
  • Not everybody has the latest and greatest.
  • Speed:
  • Reduce image usage
  • SVG/Webfonts are lower bandwitch
  • Optimize and minify CSS and JS files
  • Gzip files to eleminate unnecessary data
  • Placement:
  • Move menu to easiest reached
  • Make it simple so people can use easily
  • Forms:
  • Label move to the top when user tupe something
  • Increase fields
  • Autocomplete forms
  • Allow the user hide or not the password
  • Credit card information in one field
  • Use number pad
  • Phone number with "a" tag
  • Ads:
  • Make it small and simple
  • Stop using pop-ups/pop-overs
  • Make sure it's unobstrutive
  • Content first
