Week 02 - My notes

Undefined, NaN and infinity are reserved words.

Primitive data types: string, symbol, number, boolean, undefined, null. Any value that isn't one of thr primitive data types above is an object. These includes arrays, functions and object literals.

JS has a special operator called typeof for finding out tupe of a value.

    Variables:
  • const - when the variable will not reassigned to another value.
  • let - if variable might be reassigned.

If the variable references a non-primitive data type, such as an array, function or object, then using const will not make it immutable.

Global Scope - any variable declared outside of a block. This means it is accessible everywhere in the program. Blocks can be used to create a local scope.

Primitive data types and objects have properties and methods. Properties are information about the object or value, while methods perform an action on the object or value, either to change it or to tell us something about it.

Symbols can be used to create unique values, which helps to avoid any naming collisions.

    Array - is an ordered list of values.
  • Pop - remove the last item from an array.
  • Shift - removes the firt item
  • Push - appends a new value to the end of the array
  • Unshift - appends new item to the beginning
  • Concat - used to merge an array with one or more arrays.
  • Spreads - this can be used to spread the elements of two arrays and put them together in a new array
    Function
  • Function declaration: function keyword and is followed by the name of thefunction
  • Return Values - All functions return a value, which can be specified using the return statement,which comes after thereturnkeyword.
  • Parameters a function needs are set when the function is defined. When a function is invoked, it is provided with arguments.
  • Arrow functions can be identified by the arrow symbol,=>that gives them theirname. The parameters come before the arrow and the main body of the functioncomes after. Arrow functions are always anonymous, so if you want to refer tothem, you must assign them to a variable
  • Functions can also be given as a parameter to another function
